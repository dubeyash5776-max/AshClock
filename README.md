## 🚀 Quick Start

1. Install Rainmeter.
2. Download the latest **`.rmskin`** installer from the **Releases** page.
- **Quick Install (.rmskin):** [Download AshClock Installer](https://github.com/dubeyash5776-max/AshClock/releases/download/v1.0.0/AshClock_1.0.0.rmskin)
3. Double-click the installer.
4. Click **Install**.

Done! 🎉

## 📦 Installation

### 🚀 Option 1 (Recommended) — Install via `.rmskin`

1. Install the latest version of Rainmeter.
2. Go to the **Releases** page of this repository.
3. Download the latest **AshClock_1.0.0.rmskin** file.
4. Double-click the `.rmskin` file.
5. Click **Install** in the Rainmeter Skin Installer.

That's it! AshClock will be installed automatically. 🎉

---

### 🛠️ Option 2 — Manual Installation

1. Download or clone this repository.
2. Copy the **AshClock** folder to:

```text
Documents/Rainmeter/Skins
```

3. Right-click the Rainmeter tray icon.
4. Click **Refresh all**.
5. Load **AshClock.ini** from the Rainmeter Manage window.

Enjoy! ✨
